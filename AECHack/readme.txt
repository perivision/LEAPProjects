X3D Bim Server Extension

Converted from Collada serializer 3/15/14 for AEC Hackathon

Usage:

Install Eclipse for Java

Install BIMServer 1.1 (NOT the later releases)

Unzip archive into the BIMServer tree.

Load project

Run the project with LocalDevBimServerStarter. This will start a local server on localhost:8080

Use a browser to open localhost:8080. This should present a web page asking for login.
User: admin@bimserver.org
Pass: admin

Choose 'Add Project'

Upload your IFC file

In resulting page, click on your project's name. This should present a new page which has a 'Download' link on it. Click that.

In the drop-down, choose X3D. Your X3D file should begin downloading!
